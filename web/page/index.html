Select LEDs/Pattern:
#LEDs:

Patterns can include individual LEDs or ranges of LEDs separated by commata, e.g. "1-3,7,20-35". You can select colors via the color-picker or by clicking a color inside the color display. If you set a pattern, use a low positive integer. "#LEDs" is the total number of LEDs connected to the esp8266 and must be provided.
